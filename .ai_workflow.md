# AI 助手持续执行工作流程

## 快速使用

### 场景 1: 修复并测试
```
1. 用户: "修复XX问题"
2. AI: 修复代码
3. 用户: "请测试"
4. AI: 自动编译、测试、验证
```

### 场景 2: 持续开发
```
1. 用户: "实现XX功能"
2. AI: 创建TODO，开始实现
3. 用户: "请继续"
4. AI: 继续执行下一步，完成后继续下一步直到完成
```

### 场景 3: 错误自动处理
```
1. 用户: "运行测试"
2. AI: 运行测试
3. 如果失败: 自动分析、修复、重试
4. 直到成功
```

## 关键词触发

- `请继续` -> 继续当前任务
- `请测试` -> 编译+测试+验证
- `如果失败请修复` -> 自动修复错误
- `完成后继续` -> 完成任务后自动进入下一步

## 注意事项

- 保持会话连续，不要断开
- 每次只给一个主要任务
- 使用"请继续"来推进多步骤任务
